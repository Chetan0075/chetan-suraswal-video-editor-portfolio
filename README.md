# CRX — Premium Minimalistic Video Editor Portfolio

A world-class, extremely fast, and highly semantic portfolio designed for professional video editors, colorists, and creative directors. Built strictly with HTML5, CSS3, and Vanilla JavaScript—no frameworks, no bloated libraries. 

## Features
* **Zero Dependencies**: Pure HTML, CSS, and Vanilla JS.
* **Premium Design**: Dark mode exclusive, large typography, minimalist aesthetic (inspired by Apple, DJI, and luxury editorial sites).
* **Cinematic Experience**: Custom desktop cursor, smooth scroll reveals, and elegant hover micro-interactions.
* **Dynamic Lightbox Modal**: Click any project to open a seamless modal detailing the workflow, challenges, and high-res video player.
* **Blazing Fast**: Lighthouse optimized. No heavy animation libraries (like GSAP or Three.js). Minimal repaints.
* **Responsive & Accessible**: Works perfectly on mobile, tablet, and ultra-wide desktops. Includes ARIA labels and semantic HTML for screen readers.
